# 스마트폰 앱: 시니어 AI 말벗 & 맞춤 음악 앱 개발 로드맵

**개발 방향성**: 제공해주신 레퍼런스 이미지 구조에 맞추어, 전체 개발 공정을 8개의 파트(Part)로 깔끔하게 정리했습니다. 

---

### Part 1. 시작하기
- 개발 환경 세팅 (React + Vite)
- PWA (모바일 앱 설치) 셋업
- Github 리포지토리 연결
- 첫 화면(랜딩 페이지) UI 뼈대 생성

### Part 2. 사용자 관리 및 DB 연동
- 닉네임 설정 및 간편 로그인 페이지 생성
- Supabase (또는 Firebase) 연결 설정
- 사용자(users) 정보 테이블 생성
- 개인별 대화/음악 기록용 DB 설계

### Part 3. 시니어 맞춤형 디자인 (UI/UX)
- 시니어 전용 디자인 시스템 (큰 글씨, 고대비 색상)
- 가장 하단 큼직한 공통 네비게이션 탭 생성
- 화면 전체를 꽉 채우는 대형 터치 버튼 컴포넌트 제작
- 어르신 맞춤형 친절한 안내 문구 적용

### Part 4. AI 말벗 챗봇 (AI Chat)
- 채팅 화면(UI) 디자인
- Google Gemini API (또는 OpenAI) 연동
- '다정하고 친절한 상담사' 프롬프트 엔지니어링 
- 채팅 기록(History) 저장 및 불러오기

### Part 5. 음악 검색 및 AI 작곡 (AI Music)
- YouTube API 및 AI 음악 생성 API (예: Suno API) 분석
- 원하는 분위기의 AI 맞춤형 노래(작곡/작사) 생성 로직 구현
- 기존 유튜브 가수/노래 검색 및 스트리밍 로직
- 복잡한 유튜브 화면을 감춘 '심플 통합 음악 플레이어' UI 제작

### Part 6. 편의 기능 및 디테일 (Advanced UX)
- 키보드 없이 말로 치는 '음성 입력(마이크)' 기능 연결
- 음악 생성/로딩 시 지루하지 않게 하는 큰 로딩 애니메이션
- 버튼 터치 시 시각적 반응(클릭 이펙트) 폴리싱
- 전체 디자인 디테일 및 여백 수정

### Part 7. 앱 배포 (Deploy)
- Vercel (또는 Firebase Hosting)을 통한 서비스 배포
- 서비스 전용 도메인(URL) 추가 연결
- 실제 스마트폰 '바탕화면 홈 구동(PWA)' 정상 작동 최종 테스트

### Part 8. 마치며 (Next Steps)
- 앱 스토어(Play Store, App Store) 정식 출시 검토
- 가족/보호자와 대화 내용 공유 또는 안부 알림 기능 구상
- 유지보수 환경 및 최종 프로젝트 리팩토링 마무리
